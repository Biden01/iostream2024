2. «Сан белгісі» тапсырмасы

Шарт

Математикада функцияның sign(x) (санның таңбасы) келесідей анықталады:

sign (x) = 1, егер x > 0 болса,

sign (x) = -1, егер x < 0 болса,

sign (x) = 0, егер x = 0 болса.

Берілген x саны үшін sign(x) мән белгісін басып шығарыңыз. Бұл мәселені каскадты if... elif... else инструкциялары арқылы шешкен жөн.

Барлық тапсырмаларда input() арқылы енгізуді оқыңыз және print() арқылы жауапты шығарыңыз.
